Auto-Parts Website



An end of the semester school project to create a website and database for an exsisting auto parts store that was created within two weeks. The system handles credit card payment, assist in packing and shipping of the order, and helps to keep inventory.

• Front end created using HTML, CSS, and VS Code.
• Back end created using SQL, PHP and MariaDB on the command line.

3D aim trainer



A 3D aim trainer created in the Unity game engine. It is a simple game that allows the player to practice aiming in first person shooters. Scripts for the player movement, player actions, and enemy interactions were created using C#.
